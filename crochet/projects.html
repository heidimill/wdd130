Crochet Catalogue Projects Basic Information

Projects

The following projects are beginner approved!

Cat Beanie:

This is a perfect first crochet project. Any yarn works well.

This pattern can be found by searching "Crochet Cat Beanie" on YouTube.

The materials needed are yarn, crochet hook (matching yarn recommendation), and a darning needle.

Total time: An hour and a half but could vary based on yarn weight.

Difficulty: 2/5 Learning how to turn on rows can be diffeicult but overall there are only two stitched worked in a continuous rectangle shape.

Hacky sacks:

This pattern can be found searching hacky sack crochet pattern on google.

The materials needed are yarn, crochet hook (one size smaller than recommendation), darning needle, and dry pinto beans.

Total time: An hour

Difficulty: 4/5 Working in rounds is quite easy. Learning how to crochet an invisible decrease is a bit trickier but still doable. Crocheting around beans also provides a challenge

Mushroom pops:

This pattern is found on YouTube from "Cable and Canvas".

The materials needed are two yarn colors, crochet hook (one size smaller than recommendation), darning needle, polyfil, and safety eyes.

Total time: An hour

Difficulty: 5/5 For a brand new beginner, this pattern is quite difficult. Working around polyfil can add a level of confusion. As soon as the crafter has more experience with decreases, this pattern is more of a 3/5 difficulty level.
